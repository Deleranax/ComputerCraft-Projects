addProgressBar (x, y, width, length, color[, bcolor=gray], value, maxValue[, vertical=false])
Add a progress bar

Parameters:

  • x number Progress bar abscissa
  • y number Progress bar ordinate
  • width number Progress bar max width
  • length number Progress bar max length
  • color color Progress bar color
  • bcolor color Background color (default gray)
  • value number Current value
  • maxValue number Max value
  • vertical bool Make this bar vertical (default false)

Returns:

    number id Object ID
